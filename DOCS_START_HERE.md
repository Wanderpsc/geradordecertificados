## ✅ Validação

Antes de começar qualquer deploy:

```bash
npm run validate
```

Este comando verifica:
- ✅ Variáveis de ambiente
- ✅ Estrutura de arquivos
- ✅ Dependências
- ✅ MongoDB URI
